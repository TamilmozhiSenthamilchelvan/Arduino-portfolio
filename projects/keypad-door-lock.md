# Keypad Based Door Lock System using Arduino

## Overview
This project implements a keypad-based door lock system using Arduino.
The user enters a passcode using a keypad. If the passcode is correct,
a servo motor unlocks the door. An LCD displays system messages and a
buzzer alerts for incorrect attempts.

## Components Used
- Arduino Uno
- 4x4 Matrix Keypad
- 16x2 LCD Display
- Servo Motor
- Buzzer

## Working Principle
- User enters passcode through keypad
- LCD displays entered data
- Correct password → Servo unlocks
- Wrong password → Buzzer ON and error message
